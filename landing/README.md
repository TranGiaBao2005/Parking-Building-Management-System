# ParkSmart landing page

Đây là website giới thiệu độc lập, không được thêm vào router hoặc menu của Flutter app.

Mở `landing/index.html` trực tiếp trong trình duyệt hoặc chạy một static web server tại thư mục `landing` để xem. Khi deploy, có thể đặt landing page ở domain chính và Flutter app ở đường dẫn/domain riêng.
